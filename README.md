# Games Tier List

A simple and interactive tier list application for ranking your favorite video games.

## Features

### Tier System
Six customizable tiers for ranking games:
- **TOP 3** - Your absolute favorites
- **Obra-Prima** (Masterpiece) - Exceptional games
- **Boa Experiência** (Good Experience) - Solid games
- **Nem Fede nem Cheira** (Mediocre) - Average games
- **Nem de Graça** (Not even free) - Poor games
- **Nunca joguei** (Never played) - Unplayed games

### Interactive Features
- **Drag and Drop** - Drag games between tiers and the game pool
- **Local Storage** - Your tier list is automatically saved in the browser
- **Reset Button** - Clear all rankings and start fresh

## How to Use

1. Open `index.html` in a web browser
2. Drag game images from the pool to your desired tier
3. Rearrange games as needed - your changes are saved automatically
4. Click "Resetar Tier List" to clear all rankings

## Technology Stack

- **HTML5** - Structure
- **CSS3** - Styling with custom tier colors
- **Vanilla JavaScript** - Drag and drop functionality and local storage
